# Spring Boot Application Foundation

Фундамент для Spring Boot приложений с преднастроенным логированием в формате CEF.

### Логирование
- **Формат CEF** - структурированное логирование в стандартном формате
- **Двойное хранение** - логи записываются одновременно в:
  - Базу данных
  - Файлы на сервере

### Автоматическое обслуживание логов
Внутренний планировщик `org.springframework.scheduling` выполняет:

1. **Управление партициями БД**
   - Автоматическое создание партиций для данных аудита
   - Своевременное удаление устаревших партиций

2. **Ротация лог-файлов**
   - Архивация оперативного лог-файла `cef.log` в `tar.gz`
   - Организованное хранение исторических данных
